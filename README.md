### 安装

```
git clone https://github.com/Joeays/vms.git
//先git克隆或者下载下来，进入项目目录执行npm命令
npm install
```

#### 安装运行依赖要求：

基于 Koa 2.5.0，使用了异步函数。其中Node.js版本推荐 12.17.0、Vue 2.6.12。
